before start :</br>
1- add usernames to ./username.txt</br>
2- login to instagram account and copy all Cookies to ./info.conf [as one line ; ]</br>
3- run terminal/cmd -> java -jar jinsta.jar</br>
4- wait a few seconds for update links</br>
